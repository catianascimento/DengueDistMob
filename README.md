## Notebooks Description

 

### 1. Correlation Analysis (`01_correlation_analysis.ipynb`)

- Computes Pearson correlations between dengue time series

- Identifies maximum correlation time lags (-11 to 11 weeks)

- **Inputs:** Weekly dengue cases by traffic zone

- **Outputs:** Correlation matrices, lag analysis results

 

### 2. Spatial Network Plots (`02_spatial_network_plots.ipynb`)

- Generates spatial maps and network graphs

- Calculates network metrics (degree, centrality, clustering)

- **Inputs:** Zone coordinates, correlation matrices

- **Outputs:** Maps, shapefiles, network metrics CSV

 

### 3. Relationship Analysis (`03_relationship_analysis.ipynb`)

- Analyzes relationships between distance, mobility and correlations

- Generates scatter plots and trend analyses

- **Inputs:** Distance matrix, mobility matrix, correlation data

- **Outputs:** Relationship plots, statistical summaries
